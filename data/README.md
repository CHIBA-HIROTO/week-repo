# データ配置について

このフォルダには、Streamlitアプリで読み込むCSVファイルを配置してください。

## フォルダ構成

- `data/uecs/`: 環境データ（UECS）のCSVファイルをここに置いてください。
    - 複数のファイルがある場合、**更新日時が最も新しいファイル**が自動的に読み込まれます。
- `data/growth/`: 生育調査データのCSVファイルをここに置いてください。
    - ここにある**すべてのCSVファイル**が読み込まれ、統合されます。

## ファイル形式

従来のアプリで使用していたCSV形式と同じものを使用してください。
